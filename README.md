## Anpassungen

### Zu ändernde Inhalte

| Feld | Wo ändern | aktueller Wert |
|------|-----------|----------------|
| Foto | index.html, Zeile ~160 | placeholder.png |
| Email | index.html, Zeile ~280 | deine-email@beispiel.de |
| LinkedIn | index.html, Zeile ~287 | https://linkedin.com |
| GitHub | index.html, Zeile ~294 | https://github.com |
| Privat-Text | index.html, Zeile ~210 | [HIER DEIN PRIVATER TEXT] |
| Projekte | index.html, Zeile ~230-260 | Projekt-Beschreibungen |

### Eigenes Foto hinzufügen

1. Foto in den Ordner hochladen
2. In index.html die URL ersetzen:
   - Von: `https://via.placeholder.com/200x200/0066ff/ffffff?text=IB`
   - Zu: `mein-foto.jpg` (oder deine Bild-URL)
